#index
用户界面  
1. 在以下界面有不同响应式布局效果：
    - 基本, 大屏幕 
    - 平板横向
    - 平板纵向 
    - Iphone7+纵向
2. 元件表

| 按钮名称  | 跳转页面 / 说明 |

| ------------- | ------------- |

| 登入状态登出按钮  | 登入状态下，点击登出  |

| 登入状态设置按钮  | 个人设置页面  |
```
1	登入状态登出按钮	鼠标单击时:
  Case 1:
    设置 导航栏 为 未登入状态 逐渐 out 500ms 逐渐 in 500ms
    隐藏 登入状态点击头像或下拉菜单 逐渐 500 ms	登入状态下，点击登出
2	登入状态设置按钮	鼠标单击时:
  Case 1:
    设置 网页内容（不同的页面） 为 个人设置页面
    隐藏 登入状态点击头像或下拉菜单 向上滑动 500 ms	
3	网页内容（不同的页面）	状态改变时:
  Case 1:
    滚动到 锚点 (仅垂直)	因为只制作了一个页面，所以网页主要内容的切换依靠本动态面板实现
4	登入状态我发布的项目按钮	鼠标单击时:
  Case 1:
    设置 网页内容（不同的页面） 为 我发布的项目列表页
    隐藏 登入状态点击头像或下拉菜单 向上滑动 500 ms	
5	登入状态添加项目按钮	鼠标单击时:
  Case 1:
    设置 网页内容（不同的页面） 为 添加项目页
    隐藏 登入状态点击头像或下拉菜单 向上滑动 500 ms	
6	登入状态我开发的项目按钮	鼠标单击时:
  Case 1:
    设置 网页内容（不同的页面） 为 我开发的项目列表页
    隐藏 登入状态点击头像或下拉菜单 向上滑动 500 ms	
7	取消按钮	鼠标单击时:
  Case 1:
    隐藏 登入状态点击头像或下拉菜单 向上滑动 500 ms	
8	登入状态专业用户入口按钮	鼠标单击时:
  Case 1:
    Set 值于 condition = "pro"
    设置 网页内容（不同的页面） 为 后台页面（开发此页面请先联系易燃询问用途）
    隐藏 登入状态点击头像或下拉菜单 向上滑动 500 ms	
9		鼠标单击时:
  Case 1:
    设置 导航栏 为 未登入状态 逐渐 out 500ms 逐渐 in 500ms
    隐藏 未登入状态下拉菜单 逐渐 500 ms	
10	取消按钮	鼠标单击时:
  Case 1:
    隐藏 未登入状态下拉菜单 向上滑动 500 ms	
11	未登入状态专业用户入口按钮	鼠标单击时:
  Case 1:
    Set 值于 condition = "pro"
    设置 网页内容（不同的页面） 为 登入
    隐藏 未登入状态下拉菜单 向上滑动 500 ms	
12	未登入状态添加项目按钮	鼠标单击时:
  Case 1:
    Set 值于 condition = "addProject"
    设置 网页内容（不同的页面） 为 登入
    隐藏 未登入状态下拉菜单 向上滑动 500 ms	
13	未登入状态我发布的项目按钮	鼠标单击时:
  Case 1:
    Set 值于 condition = "myProject"
    设置 网页内容（不同的页面） 为 登入
    隐藏 未登入状态下拉菜单 向上滑动 500 ms	
14	未登入状态我开发的项目按钮	鼠标单击时:
  Case 1:
    Set 值于 condition = "myCode"
    设置 网页内容（不同的页面） 为 登入
    隐藏 未登入状态下拉菜单 向上滑动 500 ms	
15	登入按钮	鼠标单击时:
  Case 1:
    设置 网页内容（不同的页面） 为 登入
    Set 值于 condition = "null"
    隐藏 未登入状态下拉菜单 向上滑动 500 ms	
16	锚点		原型中切换页面滚动到网页最上方使用
```
